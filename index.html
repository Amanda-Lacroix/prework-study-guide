HTML

  • The body element represents the visible content shown to the user.
  • HTML Elements are case-sensitive.
  • HTML Atributes define additional information about an element and always defined in the beginning element.

GIT

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A : stages the modifications and indicates we want to add all changes
  • git commit -m : commits the changes with a commit message
  • git pull: receives a branch's modifications into the local environment as defined by user. EX. git pull origin main : from the origin (Git Repo) from the branch (main). This ensures you have the most up-to-date code before making changes.
  • git push: pushes the changes we have locally to our remote GitHub branch as defined by user. EX git push origin feature/starter-code
  • Pull Request is used to merge a coparison branch (feature branch) into the base branch(main)
